# # Bird Species Distribution & Biodiversity Analysis

## Project Overview

This project analyzes the distribution and diversity of bird species across two distinct ecosystems: **forests** and **grasslands**. Using observational bird data, the study explores how environmental factors and habitat types influence bird populations, their seasonal patterns, and overall biodiversity.

The insights gained aim to support wildlife conservation, habitat management, eco-tourism development, sustainable agriculture, and policy-making.

---

## Dataset

- **Source:** Bird_Observation_DataSet (multiple Excel sheets per administrative unit)  
- **Content:** Species observations with environmental, temporal, and location data  
- **Key Columns:**  
  - Admin_Unit_Code, Sub_Unit_Code, Site_Name, Plot_Name  
  - Location_Type (Forest/Grassland)  
  - Date, Time, Observer, Visit  
  - Species details: Common_Name, Scientific_Name, Detected counts  
  - Environmental factors: Temperature, Humidity, Sky, Wind, Disturbance  

---

## Project Structure

- **Data Cleaning & Preprocessing**  
  Handling missing values, duplicates, data type conversions, and outlier detection.

- **Exploratory Data Analysis (EDA)**  
  Understanding species distribution, habitat preferences, temporal patterns, and environmental impact.

- **SQL Queries & Views**  
  Database setup and creation of views for advanced aggregation and ranking.

- **Dashboards**  
  Interactive visualizations showcasing biodiversity, temporal trends, species activity, and environmental correlations.

---

## Key Features

- Analysis of bird observations by habitat type and administrative units  
- Identification of top species and biodiversity hotspots  
- Seasonal and temporal pattern analysis of bird activity  
- Environmental condition correlations (temperature, humidity, wind, sky)  
- Observer performance and visit impact  
- Conservation status insights for priority species

## How to Use

1. **Database Setup**  
   - Create the database and load data into the `bird` table.  
   - Run provided SQL scripts for data cleaning, transformations, and view creation.

2. **Exploratory Data Analysis**  
   - Use Jupyter notebooks or Python scripts for data preprocessing and analysis.

3. **Dashboards**  
   - Visualize results with tools like Power BI or Streamlit integrated with Plotly.
